# koa-inventor

> 交互式快速创建 `koa` 项目的 CLI 工具 🚀🚀，无需配置即可快速开发。

## 功能

1. 交互式创建 `Koa` 项目
2. 支持可配置的中间件:
   - koa-static
   - koa-router
   - koa-bodyparser
3. 支持选择包管理工具（npm、yarn、pnpm）

## 安装

```shell
npm install koa-inventor -g
```

## 使用

- 创建项目

```shell
koa-inventor create
```

- 创建模块

```shell
koa-inventor gm [moduleName]
```

- 帮助

```shell
koa-inventor -h
```
